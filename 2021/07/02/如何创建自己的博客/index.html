如何部署自己的博客(window 环境)

本地配置与安装

  1. 安装 node.js
  2. 安装 淘宝镜像
  3. 安装 hexo 框架 npm install -g hexo-cli.
  4. 查看 hexo 版本 hexo -v
  5. 创建博客目录 mkdir myblog
  6. 进入博客目录 cd myblog
  7. 初始化博客 hexo init
  8. 启动本地博客 hexo s/server, 地址:http://localhost:4000
  9. 创建一篇博客: hexo n “your blog name”
  10. 进入博客目录: cd source/_posts/,编辑自己的博客
  11. 返回 blog 根目录: cd ../..;清空一下: hexo clean;
  12. 生成博客: hexo g/generate

配置到远端(github)

  1. 创建一个新 repository,严格命名为: 你的 github 账户名.github.io
  2. 安装部署工具:npm install –save hexo-deployer-git
  3. blog 根目录下配置 config.yml 文件:
    到文件最底部#Deployment 部分修改为:
    deploy:
    type: git
    repo: 刚才新建的 repository 的地址(如:https://github.com/yd1035398155/yd1035398155.github.io)
    branch: master
  4. 部署到 github:hexo d/deploy
